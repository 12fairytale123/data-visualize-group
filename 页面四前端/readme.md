一、文件结构设计：
src/
 ├── pages/
 │    └── TeaInsightPage.tsx        # 页面主入口
 │
 ├── components/
 │    ├── TeaBarChart.tsx           # 人群偏好柱状图
 │    └── TeaWordCloud.tsx         # 关键词词云图
 │
 ├── services/
 │    └── teaApi.ts                # 数据接口适配层（支持mock/后端切换）
 │
 ├── data/
 │    └── mockTeaData.ts           # 模拟数据源（用于开发与展示）
 
 二、核心功能：
 2.1 点击左侧柱状图内的茶类型柱块，此时高亮相应柱块，右侧出现词云图。
 2.2 双击柱状图下的茶类型文本，此时柱状图高亮选中的茶类型.
 2.3 当高亮柱块时，再点击一次选中柱块回到初始模式。
 2.4 柱状图部分
 使用 ECharts 堆叠柱状图（Stacked Bar Chart）
对比三类核心人群：
Z世代
新中产
银发族
展示五类茶叶消费偏好：
绿茶
红茶
乌龙茶
果茶
花茶
2.5 词云图部分
基于 ECharts WordCloud 实现
按用户群体动态切换数据
展示消费者关注关键词与情绪倾向
情绪分类：
🟢 正向关键词（如：品质、养生、果香）
🔴 负向反馈（如：价格偏高、香精味）
⚪ 中性词汇
功能特点：
字体大小 = 词频权重
鼠标悬停显示详细信息（热度 / 情绪）
动态切换不同人群数据
